\subsection{Emphasis}\label{emphasis}

\emph{Italics} are created by bookending a word in asterisks while
\textbf{Bolds} are created by bookending a word in double asterisks.
Alternately, you can use \emph{single} or \textbf{double underscores}

If you want to show r code, without running it, put code between three
backticks

\begin{verbatim}
2 + 2 =4
ggplot(data, aes(x=x, y=y)) + geom_point()
\end{verbatim}
